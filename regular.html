4. This is a hyperlink (to open something with a click)

<a href="https://www.w3schools.com">Visit W3Schools</a>

You can use links to open another page of your website, it will look like this: 

<h2> <a href="regular.html"> 1.Regular HTML</a> </h2>
Where 'regular.html' is another file that contains another web page (needs to have <html>, <body> etc )
Remember that this file needs to be in the same folder as the one that contains the link.
